PORTFOLIO SITE (HTML/CSS/JS)

Project structure:
- index.html
- css/style.css
- js/main.js
- assets/images/posters/
- assets/images/works/
- assets/videos/motion/

How to use your own media:
1) Put poster JPG/PNG files in assets/images/posters/
2) Put design work JPG/PNG files in assets/images/works/
3) Put motion MP4 files in assets/videos/motion/
4) Open index.html and edit each card's data-src path, title, and description.

Example file names used in index.html:
- assets/images/posters/poster-01.jpg
- assets/images/posters/poster-02.jpg
- assets/images/works/work-01.jpg
- assets/images/works/work-02.jpg
- assets/videos/motion/motion-01.mp4
- assets/videos/motion/motion-02.mp4

Tip:
- Duplicate any card block in index.html to add more projects.
